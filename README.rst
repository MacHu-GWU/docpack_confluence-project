Install
------------------------------------------------------------------------------

``docpack_confluence`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install docpack-confluence

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade docpack-confluence
